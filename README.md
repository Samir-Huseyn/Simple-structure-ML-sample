# Airbnb Price Prediction Project

This project focuses on cleaning Airbnb listing data and building a Machine Learning model to predict apartment prices.

## Technologies Used

* Python
* Pandas
* Scikit-learn

## Project Workflow

### 1. Data Cleaning

* Removed unnecessary columns
* Handled missing values
* Encoded categorical features using One Hot Encoding

### 2. Machine Learning

* Train/Test Split
* Linear Regression Model
* Model Evaluation using:

  * MAE (Mean Absolute Error)
  * R2 Score

## Dataset Features

Some important features used:

* latitude
* longitude
* room_type
* minimum_nights
* availability_365
* number_of_reviews

## Model Goal

Predict Airbnb listing prices based on property information.

## Results

The model performance is evaluated using:

* Mean Absolute Error
* R2 Score
